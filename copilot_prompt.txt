## TL1 Command Structure

```
VERB-MODIFIER:[TID]:[AID]:[CTAG]::[PARAMETERS];
```

- **VERB**: Action (RTRV, ENT, DLT, ED, etc.)
- **MODIFIER**: Object type (OC12, T1, USER, etc.)
- **TID**: Target Identifier (network element ID)
- **AID**: Access Identifier (specific resource)
- **CTAG**: Correlation Tag (command sequence number)
- **PARAMETERS**: Optional command-specific parameters

## Platform Differences

**1603 SM**: Base system with IPAREA support and extended VPL operations
**1603 SMX**: Enhanced system with STS12C, POSPORT, and BLSR capabilities
